# 효모 활성화 센서(3피스 에어락)

공기가 빠져나갈 때 마다 에어락 부품이 움직이는 점을 이용하여 공기가 얼마나 빠져나가는지를 측정하여 효모의 활동량을 측정하는 센서 입니다.

## 재료
* 3D 프린터 (또는 직접 제작)
* 3피스 에어락
* NodeMCU
* 레이저 모듈
* 조도센서
* 저항
* USB 케이블
* USB 충전기
